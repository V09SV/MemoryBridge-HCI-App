# 📱 MemoryBridge – HCI Design Project

## 📌 Overview

MemoryBridge is a mobile application designed to assist elderly users in managing daily tasks, reminders, and family interactions using Human-Computer Interaction (HCI) principles.

## 🎯 Objective

To design an intuitive and accessible interface that reduces cognitive load and improves usability for elderly users.

## 🧠 Key Features

* Smart Reminders (medication, tasks)
* Daily Routine Tracking
* Family Memory Gallery
* Voice Notes
* Emergency SOS Assistance
* Accessibility Settings (font size, voice guidance)

## 🎨 Design Approach

The application is designed using **HCI principles**, including:

* Recognition over recall
* Minimal cognitive load
* Accessibility-first design
* Consistency and usability

## 🛠️ Tools Used

* Figma (UI/UX Design)
* Design Systems & Icon Libraries
